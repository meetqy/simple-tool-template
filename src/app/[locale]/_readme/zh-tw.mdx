# 簡單工具範本

一個單功能工具網站範本。專注於建構您的核心工具，同時利用現成的結構，包含國際化 (i18n) 和 SEO 最佳化。 非常適合快速部署小型、專注的實用程式。

- [Site](https://simple-tool-template.vercel.app/)
- [GitHub](https://github.com/meetqy/simple-tool-template)

## ✨ 特性

- **🌍 國際化 (i18n) 支援：** 內建對多種語言的支援，使用 `next-intl`。
- **📱 反應式設計：** 在所有裝置上無縫工作。
- **🎨 漂亮的 UI，使用 NextUI：** 現代且視覺上吸引人的使用者介面，使用 NextUI 元件庫。
- **🚀 快速且對 SEO 友善，使用 Next.js：** 利用 Next.js 框架的效能和 SEO 優勢。
- **💻 輕鬆的開發工作流程：** 流線型的開發體驗，實現快速迭代。
- **🔍 SEO 最佳化，支援多語言：** 針對搜尋引擎進行最佳化，支援多種語言的元描述和內容。 輕鬆翻譯您的工具描述以擴大覆蓋範圍。

## 🛠️ 技術堆疊

- **[Next.js](https://nextjs.org/):** 用於生產的 React 框架，提供伺服器端渲染、靜態網站產生和路由。
- **[next-intl](https://next-intl-docs.vercel.app/):** 用於管理翻譯和地區設定路由的國際化 (i18n) 函式庫。
- **[NextUI](https://nextui.org/):** 一個 React UI 函式庫，提供基於 Tailwind CSS 的預建、時尚元件。
- **[Tailwind CSS](https://tailwindcss.com/):** 一個實用程式優先的 CSS 框架，用於快速設定應用程式樣式。
- **[VSCode i18n Ally](https://marketplace.visualstudio.com/items?itemName=lokalise.i18n-ally):** 一個 VSCode 擴充功能，用於高效地一鍵翻譯 `(en/ja/ko).json` 檔案。

## 🚀 快速開始

### **克隆儲存庫：**

```bash
git clone https://github.com/meetqy/simple-tool-template.git
```

### **安裝依賴：**

```bash
pnpm install
```

### **複製環境變數：**

```bash
cp .env.example .env
```

### **執行開發伺服器：**

```bash
pnpm dev
```

## 📖 如何使用

1. **實現您的工具：** 在 `src` 目錄中新增您的工具的核心功能。

2. **翻譯內容：** 在 `messages` 目錄中更新各種語言的翻譯。 使用 VSCode i18n Ally 進行高效翻譯。

3. **最佳化 SEO：** 修改每種語言的 SEO 相關內容（元描述、標題）以提高搜尋引擎排名。

4. **自訂 UI：** 使用 NextUI 元件自訂您網站的外觀。

5. **部署您的網站：** 將您的應用程式部署到託管供應商，如 Vercel、Netlify 或 AWS。

## 📝 i18n 設定

1. **新增新語言：** 在 `messages` 目錄中建立新的語言檔案。 遵循現有的檔案結構。

2. **使用 VSCode i18n Ally：** 安裝並設定 VSCode i18n Ally 擴充功能，以便在 IDE 中快速輕鬆地進行翻譯。

3. **更新語言切換器：** 修改導航中的語言切換器以包含新增的語言。
