# シンプルなツールテンプレート

単一機能のツールウェブサイトテンプレートです。 コアツールの構築に集中しながら、国際化（i18n）とSEO最適化を備えた既製の構造を活用してください。 小規模で焦点が絞られたユーティリティを迅速に展開するのに最適です。

- [Site](https://simple-tool-template.vercel.app/)
- [GitHub](https://github.com/meetqy/simple-tool-template)

## ✨ 特徴

- **🌍 国際化（i18n）のサポート：** `next-intl`を使用した、複数の言語に対する組み込みサポート。
- **📱 レスポンシブデザイン：** すべてのデバイスでシームレスに動作します。
- **🎨 NextUIによる美しいUI：** NextUIコンポーネントライブラリを使用した、モダンで視覚的に魅力的なユーザーインターフェース。
- **🚀 高速かつSEOフレンドリーなNext.js：** Next.jsフレームワークのパフォーマンスとSEOの利点を活用します。
- **💻 簡単な開発ワークフロー：** 迅速な反復のための合理化された開発エクスペリエンス。
- **🔍 多言語サポートによるSEO最適化：** 複数の言語でのメタディスクリプションとコンテンツのサポートにより、検索エンジン向けに最適化されています。 ツールの説明を簡単に翻訳して、より幅広いリーチを実現します。

## 🛠️ 技術スタック

- **[Next.js](https://nextjs.org/):** サーバーサイドレンダリング、静的サイト生成、ルーティングを提供する、本番環境向けのReactフレームワーク。
- **[next-intl](https://next-intl-docs.vercel.app/):** 翻訳とロケールルーティングを管理するために使用される国際化（i18n）ライブラリ。
- **[NextUI](https://nextui.org/):** Tailwind CSSに基づいた、事前構築済みのスタイリッシュなコンポーネントを提供するReact UIライブラリ。
- **[Tailwind CSS](https://tailwindcss.com/):** アプリケーションのスタイリングを迅速に行うためのユーティリティファーストCSSフレームワーク。
- **[VSCode i18n Ally](https://marketplace.visualstudio.com/items?itemName=lokalise.i18n-ally):** `i18n.json`ファイルの効率的なワンクリック翻訳のためのVSCode拡張機能。

## 🚀 クイックスタート

### **リポジトリをクローンする：**

```bash
git clone https://github.com/meetqy/simple-tool-template.git
```

### **依存関係をインストールする：**

```bash
pnpm install
```

### **環境変数をコピーする：**

```bash
cp .env.example .env
```

### **開発サーバーを実行する：**

```bash
pnpm dev
```

## 📖 使い方

1. **ツールを実装する：** `src`ディレクトリ内にツールのコア機能を追加します。

2. **コンテンツを翻訳する：** `messages`ディレクトリ内のさまざまな言語の翻訳を更新します。 効率的な翻訳のためにVSCode i18n Allyを使用します。

3. **SEOのために最適化する：** 検索エンジンのランキングを向上させるために、各言語のSEO関連コンテンツ（メタディスクリプション、タイトル）を変更します。

4. **UIをカスタマイズする：** NextUIコンポーネントを使用して、ウェブサイトのルックアンドフィールをカスタマイズします。

5. **サイトをデプロイする：** アプリケーションをVercel、Netlify、AWSなどのホスティングプロバイダーにデプロイします。

## 📝 i18nセットアップ

1. **新しい言語を追加する：** `messages`ディレクトリ内に新しい言語ファイルを作成します。 既存のファイル構造に従ってください。

2. **VSCode i18n Allyを使用する：** IDE内で迅速かつ簡単に翻訳を行うために、VSCode i18n Ally拡張機能をインストールして構成します。

3. **言語スイッチャーを更新する：** 新しく追加された言語を含めるように、ナビゲーションの言語スイッチャーを変更します。
