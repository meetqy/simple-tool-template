# 简单工具模板

一个单功能工具网站模板。专注于构建您的核心工具，同时利用现成的结构，包含国际化 (i18n) 和 SEO 优化。 非常适合快速部署小型、专注的实用程序。

## ✨ 特性

- **🌍 国际化 (i18n) 支持：** 内置对多种语言的支持，使用 `next-intl`。
- **📱 响应式设计：** 在所有设备上无缝工作。
- **🎨 漂亮的 UI，使用 NextUI：** 现代且视觉上吸引人的用户界面，使用 NextUI 组件库。
- **🚀 快速且对 SEO 友好，使用 Next.js：** 利用 Next.js 框架的性能和 SEO 优势。
- **💻 轻松的开发工作流程：** 流线型的开发体验，实现快速迭代。
- **🔍 SEO 优化，支持多语言：** 针对搜索引擎进行优化，支持多种语言的元描述和内容。 轻松翻译您的工具描述以扩大覆盖范围。

## 🛠️ 技术栈

- **[Next.js](https://nextjs.org/):** 用于生产的 React 框架，提供服务器端渲染、静态站点生成和路由。
- **[next-intl](https://next-intl-docs.vercel.app/):** 用于管理翻译和区域设置路由的国际化 (i18n) 库。
- **[NextUI](https://nextui.org/):** 一个 React UI 库，提供基于 Tailwind CSS 的预构建、时尚组件。
- **[Tailwind CSS](https://tailwindcss.com/):** 一个实用程序优先的 CSS 框架，用于快速设置应用程序样式。
- **[VSCode i18n Ally](https://marketplace.visualstudio.com/items?itemName=lokalise.i18n-ally):** 一个 VSCode 扩展，用于高效地一键翻译 `i18n.json` 文件。

## 🚀 快速开始

### **克隆仓库：**

```bash
git clone https://github.com/meetqy/simple-tool-template.git
```

### **安装依赖：**

```bash
pnpm install
```

### **复制环境变量：**

```bash
cp .env.example .env
```

### **运行开发服务器：**

```bash
pnpm dev
```

## 📖 如何使用

1. **实现您的工具：** 在 `src` 目录中添加您的工具的核心功能。

2. **翻译内容：** 在 `messages` 目录中更新各种语言的翻译。 使用 VSCode i18n Ally 进行高效翻译。

3. **优化 SEO：** 修改每种语言的 SEO 相关内容（元描述、标题）以提高搜索引擎排名。

4. **自定义 UI：** 使用 NextUI 组件自定义您网站的外观。

5. **部署您的站点：** 将您的应用程序部署到托管提供商，如 Vercel、Netlify 或 AWS。

## 📝 i18n 设置

1. **添加新语言：** 在 `messages` 目录中创建新的语言文件。 遵循现有的文件结构。

2. **使用 VSCode i18n Ally：** 安装并配置 VSCode i18n Ally 扩展，以便在 IDE 中快速轻松地进行翻译。

3. **更新语言切换器：** 修改导航中的语言切换器以包含新添加的语言。
