# KD和KDA计算器

## 什么是KD和KDA？

**KD（击杀/死亡比）**和**KDA（击杀/死亡/助攻比）**是许多竞技型多人在线游戏中用于评估玩家表现的重要指标。这些比率通过比较玩家获得的击杀数、死亡数和助攻数来反映玩家在游戏中的有效性。

### KD（击杀/死亡比）

**击杀/死亡比（KD）**衡量玩家在尽量减少死亡次数的同时获得击杀的能力。它是通过将击杀数除以死亡数来计算的。更高的KD比表明玩家更有效率。

**公式：**

`KD比 = 击杀数 / 死亡数`

### KDA（击杀/死亡/助攻比）

**击杀/死亡/助攻比（KDA）**是一个更全面的指标，它除了击杀数和死亡数之外还包括助攻数。它是通过将击杀数和助攻数相加，然后除以死亡数来计算的。这个比率能更好地理解玩家对团队的整体贡献。

**公式：**

`KDA比 = (击杀数 + 助攻数) / 死亡数`

## 如何使用KD和KDA计算器

1. 在“击杀数”输入框中输入击杀数。
2. 在“死亡数”输入框中输入死亡数。
3. 在“助攻数”输入框中输入助攻数。
4. 点击“计算”按钮查看结果。
5. 结果将显示KD比和KDA比。
