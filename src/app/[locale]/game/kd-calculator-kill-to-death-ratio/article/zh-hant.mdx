# KD與KDA計算器

## 什麼是KD與KDA？

**KD（擊殺/死亡比）**與** KDA（擊殺/死亡/助攻比）** 是許多競技型多人線上遊戲中用來評估玩家表現的重要指標。這些比率藉由比較玩家獲得的擊殺數、死亡數和助攻數來反映玩家在遊戲中的有效性。

### KD（擊殺/死亡比）

**擊殺/死亡比（KD）** 衡量玩家在盡量減少死亡次數的同時獲得擊殺的能力。它是透過將擊殺數除以死亡數來計算的。較高的KD比表示玩家更有效率。

**公式：**

`KD比 = 擊殺數 / 死亡數`

### KDA（擊殺/死亡/助攻比）

**擊殺/死亡/助攻比（KDA）** 是一個更全面的指標，它除了擊殺數和死亡數之外還包含助攻數。它是透過將擊殺數和助攻數相加，然後除以死亡數來計算的。這個比率能更好地理解玩家對團隊的整體貢獻。

**公式：**

`KDA比 = (擊殺數 + 助攻數) / 死亡數`

## 如何使用KD與KDA計算器

1. 在「擊殺數」輸入欄位中輸入擊殺數。
2. 在「死亡數」輸入欄位中輸入死亡數。
3. 在「助攻數」輸入欄位中輸入助攻數。
4. 點擊「計算」按鈕查看結果。
5. 結果將顯示KD比和KDA比。
