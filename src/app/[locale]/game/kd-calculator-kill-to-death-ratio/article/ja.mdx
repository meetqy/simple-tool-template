# KDとKDA計算機

## KDとKDAとは？

**KD（キル対デス比）**と**KDA（キル/デス/アシスト比）**は、多くの対戦型マルチプレイヤーゲームにおいて、プレイヤーのパフォーマンスを評価するために用いられる重要な指標です。これらの比率は、プレイヤーが獲得したキル数、デス数、アシスト数を比較することで、ゲームにおけるプレイヤーの有効性を示します。

### KD（キル対デス比）

**キル対デス比（KD）**は、デス数を最小限に抑えながらキルを獲得するプレイヤーの能力を測る指標です。キル数をデス数で割って計算します。KD比が高いほど、より効果的なプレイヤーであることを示します。

**計算式：**

`KD比 = キル数 / デス数`

### KDA（キル/デス/アシスト比）

**キル/デス/アシスト比（KDA）**は、キル数とデス数に加えてアシスト数も考慮した、より包括的な指標です。キル数とアシスト数を合計し、デス数で割って計算します。この比率は、チームへのプレイヤーの全体的な貢献度をより深く理解するのに役立ちます。

**計算式：**

`KDA比 = (キル数 + アシスト数) / デス数`

## KDとKDA計算機の使い方

1. 「キル数」入力欄にキル数を入力します。
2. 「デス数」入力欄にデス数を入力します。
3. 「アシスト数」入力欄にアシスト数を入力します。
4. 「計算」ボタンをクリックして結果を確認します。
5. KD比とKDA比が表示されます。
