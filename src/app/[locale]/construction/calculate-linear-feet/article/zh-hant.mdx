# 線性英尺計算方法：簡易指南與計算器

## 什麼是線性英尺？

**線性英尺 (linear foot)** 是一個長度單位，使用英尺作為測量單位。它用於測量物體的直線長度，在建築、木工等材料長度至關重要的領域中常用。與測量面積的平方英尺 (square foot) 不同，線性英尺僅測量長度。

## 如何計算線性英尺？

要計算線性英尺，需要知道材料的寬度（英寸）和總面積（平方英尺）。線性英尺的計算公式如下：

```plaintext
線性英尺 = (平方英尺 * 12) / 寬度（英寸）
```

### 例子

假設有一塊材料，寬度為 6 英寸，面積為 120 平方英尺。使用公式計算：

```plaintext
線性英尺 = (120 * 12) / 6 = 240 線性英尺
```
