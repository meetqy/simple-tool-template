# 線形フィートの計算方法：簡単なガイドと計算機

## 線形フィートとは？

**線形フィート (linear foot)** とは、長さの単位で、フィートを使って測定します。物体の直線の長さを測る際に使用され、建設、木工など、材料の長さが重要な分野で一般的に使用されます。面積を測定する平方フィート (square foot) とは異なり、線形フィートは長さのみを測定します。

## 線形フィートの計算方法

線形フィートを計算するには、材料の幅（インチ）と総面積（平方フィート）を知る必要があります。線形フィートの計算式は以下のとおりです。

```plaintext
線形フィート = (平方フィート * 12) / 幅（インチ）
```

### 例

幅6インチ、面積120平方フィートの材料があると仮定しましょう。公式を使用すると、以下のようになります。

```plaintext
線形フィート = (120 * 12) / 6 = 240 線形フィート
```

### 線形フィートはどのように計算しますか？

線形フィートは、以下の公式を使用して計算できます。

