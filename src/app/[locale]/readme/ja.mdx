## KDCalculator.org KDAツールの使い方

KDA計算機を使うのはとても簡単です。以下の手順に従ってください。

1. **統計情報を入力：**以下のフィールドに、ゲーム内の統計情報を入力します。

   - **キル：**倒した敵の数。
   - **デス：**倒された回数。
   - **アシスト：**敵を倒すのを手伝った回数。

2. **計算をクリック：**「計算」ボタンをクリックすると、KDA比率がすぐに生成されます。

3. **結果を分析：**計算されたKDA比率を使って、自分のパフォーマンスを評価し、改善すべき点を見つけます。

   _より詳細な分析が必要な場合は、様々なゲームモードを選択することが出来ます。_
