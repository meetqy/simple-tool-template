## 如何使用 KDCalculator.org KDA 工具

使用我們的 KDA 計算機超級簡單。只要跟著這些步驟：

1.  **輸入你的數據：** 在以下欄位填入你的遊戲數據：

    - **擊殺 (Kills)：** 你幹掉敵人的數量。
    - **死亡 (Deaths)：** 你被幹掉的次數。
    - **助攻 (Assists)：** 你協助幹掉敵人的次數。

2.  **按下計算：** 點擊「計算」按鈕，馬上產生你的 KDA 比率。

3.  **分析你的結果：** 利用算出來的 KDA 比率來評估你的表現，找出可以改進的地方。
